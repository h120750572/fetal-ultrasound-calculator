## ✨ 新增功能：CRL計算

### 📏 CRL（頭臀長）功能特色
- **適用範圍**：懷孕6-16週的早期妊娠
- **輸入參數**：CRL（頭臀長）測量值，單位毫米(mm)
- **輸出結果**：估計懷孕週數（週和天）
- **計算基礎**：基於Robinson公式，是早期妊娠準確的測量估算方法
- **數值範圍**：5-85毫米（自動驗證）

### 🎯 使用方式
1. **選擇CRL計算**：點擊「CRL計算 (6-16週)」標籤頁
2. **輸入測量值**：在頭臀長(CRL)欄位輸入測量值（毫米）
3. **點擊計算**：系統自動驗證並計算結果
4. **查看結果**：顯示估計懷孕週數（週+天）和總天數

### 📊 計算示例
- **輸入**：CRL = 30mm
- **輸出**：9週5天 (9.7週)，總天數68天

## 🔄 完整功能概覽

### Tab 1: BPD + AC計算 (18-40週)
- **適用**：中晚期妊娠（18-40週）
- **輸入**：BPD（雙頂徑）+ AC（腹圍）
- **輸出**：估計胎兒重量 + 估計懷孕週數

### Tab 2: CRL計算 (6-16週) 🆕
- **適用**：早期妊娠（6-16週）
- **輸入**：CRL（頭臀長）
- **輸出**：估計懷孕週數
